介绍

博德之门三部曲最初是由Bardez制作的模组，目的是为了实现两大目标：使用更加强化的博德之门II游戏引擎来进行博德之门与资料片剑湾传奇，并且使博德之门、剑湾传奇、安姆阴影和巴尔王座之间的事件产生连续性。博德之门三部曲-WeiDU更进一步的将这个模组转化为WeiDU格式，使其提高了和其它模组之间的兼容性。

而BGT-WeiDU大致上仍然忠于原始版本的博德之门与剑湾传奇，并且加入了一些新功能来满足上述目标。


  • 博德之门使用博德之门II的引擎： BGT-WeiDU是一个安装在博德之门II：巴尔王座的模组。因此，可以使用博德之门II的新图像、更好的解析度以及改善的介面来体验博德之门I的故事。
  • 错误修正：BGT-WeiDU包含一个内建的修正包，修正了博德之门与剑湾传奇中的大量错误。这里可以找到一个修复清单。
  • 从博德之门到安姆阴影之间的连续性：BGT-WeiDU使用了一个动态系统来将玩家在博德之门结尾时传送到安姆阴影。并且在玩家告诉贝尔特已经打败了沙洛佛克之后使用了一个过场的理由，而NPC们将会有条件的汇入到安姆阴影，这取决于你带领他们在博德之门中所经历的命运。
  • 与其它模组的兼容性全面提升：BGT-WeiDU兼容于大量模组安装时的最新版本模组，以及其他许多由爱好者制作的模组。这可以让玩家扩展博德之门的世界，到更多的地点探险。这里有一份兼容性列表。
  • 系统日志项目：BGT-WeiDU利用了完整的任务日志系统，并确保博德之门所有的日志项目在传送到安姆阴影之前删除。这可以防止日志爆满，否则这样通常会很难找到日志中的重要信息。
  • 地图注记：地图注记的出现有助于玩家了解目前的位置。BGT-WeiDU附带一些已经内建的地图注记，以帮助玩家能迅速找到特定的建筑物，而不是每到一个特定的地区就像疯狂的小鸡一样四处进出每一间房子。

需求

BGT-WeiDU需要下列：

  • 博德之门和剑湾传奇的自定义安装(所有框打上钩)，再安装5512补丁

    • 或

  • 博德之门：最初的英雄传奇的自定义安装(所有框打上钩)

    • 此安装必须是“纯净的”安装，不应该安装任何模组。

    如果你在安装过程中收到没有足够硬碟空间的警告，并且确认过你其实拥有足够的硬碟空间，那么可以忽略这个警告。

    以及

  • 完全安装博德之门II：安姆阴影(SoA)和博德之门II：巴尔王座(ToB)，再安装26498补丁

    • 这个步骤可能有必须预先安装的模组。请参阅常见问题解答了解详情。

    及

  • 安装过程至少需要3GB以上的可用硬碟空间。不包含已独立安装的博德之门和剑湾传奇，BGT-WeiDU需要使用2.5GB的空闲硬碟空间。为了使用虚拟记忆体，建议至少需要4GB以上的可用硬碟空间。

安装和移除安装

安装

    如果你安装的BGT-WeiDU只是mega模组的一部分，在属于mega模组的所有模组之后安装这个模组。请参阅常见问题解答了解更多详情。

  1. 解压缩BGT1##-Install.rar到你的博德之门II安装目录

    2. 确保你的光碟机中没有博德之门相关的CD、DVD或是虚拟镜像，或是虚拟光碟。如果做不到这一点可能会导致安装过程中发生错误。

    我们建议你在安装过程中关闭防毒软体，间谍软体扫瞄以及其他背景中运行的文件扫描。这些程式已知会在某些情况下干预安装过程。

  2. 运行在博德之门安装目录下的Setup-BGT.exe，并且依照安装提示。

    3. 安装程式会检查你输入的BG1安装目录是否包含完全安装的博德之门与剑湾传奇。如果安装程式必须中止，并使你所安装的BG1达到安装要求，你可以放心的关闭安装视窗或是输入Ctrl + C三次。

  3. 运行博德之门II - 点击博德之门按钮，然后按新游戏将开始博德之门I

    4. 你可以移除博德之门I和剑湾传奇。如果你以后有需要想重新安装BGT-WeiDU，就需要保留博德之门I和剑湾传奇。

  4. 安装后，你可能会想要启动博德之门与剑湾传奇的配乐，以及安装自订的图形使用介面。请参考安装后设置Post-installation文件的详细说明。

反安装和重新安装

  1. 运行在博德之门安装目录下的Setup-BGT.exe，并且依照安装提示。
  2. 卸载提示信息出现后，备份的dialog.tlk可在\bgt\dialog.bak目录下找到

自动安装(专业用户)

  1. 将BGT1##-Install.rar解压缩到博德之门II的安装目录下

    2. 确保你的光碟机中没有博德之门相关的CD、DVD或是虚拟镜像，或是虚拟光碟。如果做不到这一点可能会导致安装过程中发生错误。

    我们建议你在安装过程中关闭防毒软体，间谍软体扫瞄以及其他背景中运行的文件扫描。这些程式已知会在某些情况下干预安装过程。

  2. 前往命令行，定位到你的博德之门II安装目录，并且以下列的格式来执行Setup-BGT.exe。或者将这个格式放到批次档或执行……开始选单中的对话。

    3. Setup-BGT.exe --force-install 0 --skip-at-view --language X --args-list [flags] [path]

    Parameters:

    X

    0 - English(英语)
    1 - International English(国际英语)
    2 - Castilian/Spanish(西班牙语)
    3 - French(法语)
    4 - German(德语)
    5 - Italian(义大利语)
    6 - Korean(韩语)
    7 - Polish(波兰语)
    8 - Russian(俄语)
    9 - Simplified Chinese(简体中文)
    10 - Traditional Chinese(繁体中文)
    11 - Czech
    [flags]

    b或B - 隐藏biffing选项和biff所有新资源的提示（优先度高于'o'标识）

    o或O - 隐藏biffing选项和biff所选的资源的提示（安装mega-modification类mod时推荐）

    s或S - 禁止在结束安装后显示安装后设置说明

    p或P - 预设的博德之门与剑湾传奇目录完整路径；如果路径是无效的，安装程序将会提示有效的路径。
    [path] 引号中的博德之门与剑湾传奇目录完整路径，与[flags]中的p或P一起使用

    例如：

    Setup-BGT.exe --force-install 0 --skip-at-view --language 0 --args-list sp "C:\Program Files\Black Isle\Baldur's Gate"

    这种设定以英语来安装BGT-WeiDU，禁止在结束安装后显示安装后设置说明，并且将博德之门与剑湾传奇安装目录预设为C:\Program Files\Black Isle\Baldur's Gate。

    安装程式会检查你输入的博德之门I安装目录是否包含完全安装的博德之门与剑湾传奇。如果你指定的目录无效或是包含不正确安装的博德之门与剑湾传奇，安装程式将会提示你输入正确的目录。

    如果安装程式必须中止，并使你所安装的博德之门I达到安装要求，你可以放心的关闭安装视窗或是输入Ctrl + C三次。

博德之门三部曲与博德之门II其它模组的相容性如何呢？

Bardez的博德之门三部曲--BGT_MK3_112，成为了被称为全景(The Big Picture)的mega 模组中的一部份，全景将许多BG2的大型模组结合在一起(最黑暗的一天、梭巴的阴影以及饱受折磨的灵魂)，为了使玩家能同时享受所有的主要模组。随后，使用这个平台的TeamBG 模组黑暗剑湾转换为特别为博德之门三部曲所制作的模组，以及北部剑湾、埋骨丘的秘密等模组也是如此，使得被称为BP-BGT的庞大的模组集合体因而极大的拓展了博德之门的世界。

如今，BP-BGT是一种过时的称呼，而mega 模组是指至少安装了下列模组两个以上：博德之门三部曲-WeiDU(BGT-WeiDU)、最黑暗的一天(TDD)、梭巴的阴影(SOS)、饱受折磨的灵魂(TS-BPversion)、检查身体(CtB)、恐怖地带(RoT)、无尽的旅程(NeJ)以及全景(BP)。


如果Bardez制作了博德之门三部曲，那为什么还需要博德之门三部曲-WeiDU(BGT-WeiDU)呢？

凑合在一起的大量模组影响了博德之门II的各个方面，而且由不同作者制作的模组也产生了许多兼容性问题。会出现这种状况是因为同样的文件被修改和覆写，这代表着一种改变取代了另一种，而原本的改变则永远消失了。此外，大型模组通常会带来负载的错误，因而很难找到正确的文件以及修复。

你可能也猜想得到像博德之门三部曲这样的大型模组，增加了博德之门与剑湾传奇大部分的素材到博德之门II中，这需要极大容量的安装程式包，有些玩家可能由于老旧的网路连线必须枯燥而缓慢的下载大容量的档案。BGT-WeiDU就是为了解决这些问题，藉由将安装过程转换为Weimer Dialogue Utilities (WeiDU)平台，这个强大的工具可以解决兼容性问题，而且在修复错误时又快又有效率。BGT-WeiDU同时也从原始安装的博德之门与剑湾传奇中提取了许多资源，使需要下载的档案容量缩小许多。


你说BGT-WeiDU大致上仍然忠于原始版本的博德之门与剑湾传奇。那么这个模组与原始版本的博德之门与剑湾传奇之间有哪些主要的改变呢？

BGT-WeiDU加入了一些新功能，其中有许多功能利用了博德之门II引擎的优势与WeiDU的兼容性特点。在引言的部份有新功能的详细介绍。

此外，也修复了许多博德之门与剑湾传奇原有的错误。这里可以找到一份纳入BGT-WeiDU修复范围的清单。


BGT-WeiDU是如何将博德之门与剑湾传奇加入到我的博德之门II游戏中？

BGT-WeiDU转换了博德之门与剑湾传奇的资源，并且安装到你的博德之门II目录中。从各个方面看来，这种转换大致上是忠于原作的。从技术方面来看，这个模组的安装程式从你目前已有的博德之门与剑湾传奇中直接汇入了核心资源的副本。因此，你原有的博德之门与剑湾传奇完全不会受到影响。你可以在安装BGT-WeiDU之后选择反安装博德之门与剑湾传奇，以便节省一些硬碟空间。


我想要安装BGT-WeiDU。我应该纯净备份已安装的博德之门、剑湾传奇、安姆阴影以及巴尔王座吗？

拥有一个安装好的博德之门与剑湾传奇纯净备份是相当重要的，并且不能包含任何模组(包括Baldurdash fixes)。如果无法做到这一点，在安装BGT-WeiDU时将会产生许多不可预料的问题。

对于安姆阴影与巴尔王座来说，你可能已经预先安装了模组。详见下文关于与BGT-WeiDU一起使用其它模组的详细介绍。


BGT-WeiDU的安装过程需要耗费多少时间？

安装时间取决于你的电脑系统性能，但大略估计可能需要20分钟到1小时。你将会收到提示通知你在安装过程中发生了什么。你也可以在安装过程结束时收到提示。


我已经安装了BGT-WeiDU。我要如何从烛堡开始博德之门的冒险？

当你启动博德之门II后，你将会进入选择安姆阴影或是巴尔王座的主选单。请注意，安姆阴影的按钮已经改为博德之门的按钮。点击这个按钮，并且选择单人游戏，然后开始新游戏。这将会让你进入人物创造界面，你将可以创造一个等级1的人物，或是汇入一个现有的人物。你的人物将会在烛堡开始冒险。


在安装了BGT-WeiDU之后，我还能从安姆阴影或是巴尔王座开始游戏吗？

当然可以。

关于安姆阴影，点击主选单的博德之门按钮，然后选择单人游戏。你将会发现教学指南的按钮改为一个"BG2新游戏"的按钮。点击这个按钮，并且选择对话框上的BG2新游戏按钮进入人物创造介面，你可以创造一个拥有BG2序章适当经验值的人物，或是汇入一个现有的人物。当完成时，你将会在艾瑞尼卡斯的地牢开始冒险。

开始一个巴尔王座新游戏的方法并没有改变。


BGT-WeiDU中的故事连续性如何运作？

如果你从烛堡开始一个新游戏，你可以游遍原始的博德之门世界，直到打败沙洛佛克为止。剑湾传奇的区域只要你可以通往巨龙桥口的北方就能开启。打败沙洛佛克不会让游戏结束，可以让游戏继续。向公爵府的贝尔特谈话，他将会给你一个新任务。这时将会出现一个你被影贼袭击的过场动画，并且将你带到艾瑞尼卡斯的地牢。

此外，可以出现在博德之门与安姆阴影的可加入的NPC，可以有条件的保留他们的属性传送到安姆阴影，这取决于你在博德之门中离开他们后的命运。艾德温、爱蒙、贾西拉、明斯克以及维康妮亚将可以保留他们在博德之门时的属性和外观，如果他们在打败沙洛佛克后与贝尔特谈话时还在队伍中并且活着的话。否则，他们将会以未安装BGT-WeiDU的安姆阴影原有的状态出现。


博德之门与剑湾传奇本身拥有许多的错误。在这种转换下，这些错误是否仍然存在？

BGT-WeiDU试着解决存在于博德之门与剑湾传奇最终版本中最严重的错误。这里可以找到一份纳入BGT-WeiDU修复范围的清单。


在反安装BGT-WeiDU后，我是否需要还原dialog.tlk？

不，你不需要这么做。然而如果你不这么做，当你开始安装其它模组并且使用不同情况下的游戏存档可能会有问题。反安装 BGT-WeiDU最安全的方式是将dialog.tlk还原以避免这些问题。


为什么在反安装BGT-WeiDU之后，我要手动还原dialog.tlk？

BGT-WeiDU对你的博德之门II的dialog.tlk档案造成了许多改变，这个档案包含了游戏中所有的文字。由于WeiDU的限制以及BGT-WeiDU安装程式设计时的方式，会在安装或反安装过程结束时产生修改过的dialog.tlk档案。这将无法使批次档运作来自动复原你的 dialog.tlk。因此，唯一的方法是将备份的dialog.tlk档手动还原覆盖原来的档案位置。


我能安装其它的模组和BGT-WeiDU来开始游戏吗？

是的。BGT-WeiDU不只是保留原有内容，而且还提高了和mega 模组的兼容性。对于其它的模组来说，没有人能保证每个模组都能和其它模组兼容，BGT-WeiDU的兼容性也不例外。然而，BGT-WeiDU与其它模组的兼容性问题正在积极地分析中。这里有一份BGT-WeiDU的兼容性清单。最新版的兼容性清单可以在网路上找到。


我如何与Mega 模组一起安装BGT-WeiDU？

Mega 模组的安装顺序是有相当多变化的。因此，这个说明文件不会告诉你安装这些模组的顺序。然而有一个论坛致力于Mega 模组安装，你可以在这里连结论坛网址。


我如何与其它模组一起安装BGT-WeiDU？

以一般惯例来说，应该先安装BG2 Fixpack，如果你想要使用这个修正包，接着安装Mega 模组、BGT-WeiDU，然后才是其它模组。具体的安装顺序是不断变化的，因此无法给予更详细的建议。

如果你想要玩BGT-WeiDU和其它模组，但不包括Mega 模组的话，先安装BG2 Fixpack，其次是BGT-WeiDU，然后才是其它的模组。

记得小心兼容性问题。在这里列出目前BGT-WeiDU与其它模组之间的兼容性。其它模组彼此之间的兼容性应该询问各个模组的作者。


我想要翻译这个模组。我应该怎么做呢？

想要将BGT-WeiDU翻译成不同的语言，请按照下列的步骤：

  • 前往你的BGT-WeiDU安装目录，并且切换到BGT\Language 目录
  • 复制一个English目录并且改名为想要翻译的语言。例如：

    • 英语(English ) -> 西班牙语(Castiliano)

  • 在你改名的目录中，将目前所有的档案依照下列方式翻译：

    • .tra files - 每一行都以@ = 作为开头，翻译的文本要在两个~或"符号之间

      • @1001 = ~English~ -> @1001 = ~Castiliano~

      @-1001 = "English" -> @-1001 = "Castiliano"

    • PreSetup、EndSetup、和 Uninstall 目录 – 都是必须翻译的.txt文件
  • 切换到BGT\Help目录
  • 复制[English]BGTReadme.htm and [English]postinstall.htm，，并且改名为想要翻译的语言

    • [English]BGTReadme.html -> [Castilian]BGTReadme.html

    [English]postinstall.html -> [Castilian]postinstall.html

  • 翻译BGTReadme.htm的1，2，3，4，5，和9等部分，以及所有的postinstall.htm
  • 切换到BGT\Worldmap\English目录，复制worldmap.tra并且翻译其中内容

    • ###视为空格，但要确定你目前留下了###。如果你需要一个空格，将它们改为###。

    @1001 = ~English###English~ -> @1001 = ~Castiliano###Castiliano~

    @-1001 = "English###English" -> @-1001 = "Castiliano###Castiliano"

  • 将你改名后的目录, readme档案、postinstall.htm和worldmap.tra打包放在同一个资料夹中
  • 可以将这个档案发布在模组论坛，或是发送私人讯息(PM)给 Spellhold Studios forums的Ascension64，他将会给你一个电子信箱来发送翻译好的档案
  • 等待下一个版本加入新的语言

WeiDU试图删除的档案不存在，并且询问是否重新安装模组。

你可能没有纯净安装的博德之门与剑湾传奇或是博德之门：巴尔王座。

  • 当安装程式提示重新安装时，退出安装程式
  • 依照这个文件的"安装与反安装"部分所述，将dialog.tlk还原
  • 你刚刚将安装过程重置。不要马上尝试重新安装。
  • 尝试重新安装没有任何模组的博德之门与剑湾传奇或博德之门：巴尔王座，并且重新安装BGT-WeiDU。
  • 如果你有安装其它的模组，试着反安装这些模组，并且重新安装BGT-WeiDU。
  • 如果你仍然碰到问题，将位于你的BG2安装目录中的SETUP-BGT.DEBUG和SETUP-BGTDOS.DEBUG(如果存在的话)档案发表到模组论坛。

视窗给我一个关于_WeiDU.exe或tis2bg2.exe的异常错误讯息

DOS在某些情况下重复提示的错误 "WeiDU: ZLIB: Warning! Problem decompressing block."

如果你没有按照安装部分的第二步骤将会发生这些已知的问题。

  • 等待安装结束。这是相当重要的。依照"反安装与重新安装"部分所述来反安装BGT-WeiDU。
  • 还原dialog.tlk。
  • 正确的按照安装步骤指示。

小型安装错误

在安装过程中，我不断收到"你的光碟机中没有光碟"的错误。

如果你没有按照安装步骤的步骤2就会发生这种情况。

  • 如果你尚未安装，或是你在要求输入你的博德之门与剑湾传奇目录时检查光碟后发觉这个安装程式直接停止了，立刻中止安装程式。
  • 当你看到这里，如果你在要求输入博德之门与剑湾传奇目录后安装程式都会停止，按下"取消"直到所有提示出现，并且使安装程式结束。当安装程式结束后，反安装BGT-WeiDU并且将dialog.tlk还原。
  • 正确地按照安装程式的步骤，并且没有省略步骤2。
